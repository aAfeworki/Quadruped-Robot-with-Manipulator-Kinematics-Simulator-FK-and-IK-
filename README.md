# Quadruped-Robot-with-Manipulator-Kinematics-Simulator-FK-and-IK-
Interactive Python-based simulators for a quadruped robot with a manipulator, including forward kinematics, inverse kinematics with a fixed base, and inverse kinematics with a floating base.

This repository provides interactive simulators for a quadrupedal robotic manipulator system, implemented in Python using numpy, matplotlib, and tkinter.
It demonstrates both forward and inverse kinematics, making it a valuable tool for exploring the motion, posture, and control of legged robots.


🚀 Features


Forward Kinematics 


Control the biped manipulator robot’s joint angles interactively using sliders to visualize the resulting pose of the robot.

Solve joint angles for a desired foot position while keeping the base fixed.
